# osudirect-desktop

Desktop app for osu.direct.

## Features

*   Provides desktop integration for osu.direct.
*   In-game overlay for seamless interaction.

## Development

To get started with development:

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the application:
    ```bash
    npm start
    ```

## Building for Production

To build the application for production, use the following command:

```bash
npm run compile
```

This will create a distributable package in the `release` directory.

## Technologies Used

*   [Electron](https://www.electronjs.org/)
*   [asdf-overlay](https://github.com/storycraft/asdf-overlay)
*   [TypeScript](https://www.typescriptlang.org/)
*   [Node.js](https://nodejs.org/)
